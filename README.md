# Installing

Run with `pipx install .` inside the project to install to your machine.

# Config

conf.toml

```toml
[wiki]
url = "https://demo.bookstack.com"
path = "/home/user/notes/"

[wiki.excluded]
shelves = ["private"]
```

## Commmands

### Local

Updates any local changes to the remote.

### Remote

Updates any remote changes to the local store.
